# 🌡️ Temperature Converter

## 📋 Project Information
- **Task:** Level 2, Task 4
- **Internship:** Cognifyz Software Development Program
- **Language:** Python 3

## 🎯 Objective
Build a temperature converter program that enables users to input temperatures and choose the conversion direction between Fahrenheit and Celsius.

## ✨ Features

### Core Features
- ✅ Convert **Celsius to Fahrenheit** (°C → °F)
- ✅ Convert **Fahrenheit to Celsius** (°F → °C)
- ✅ Input validation (handles invalid entries)
- ✅ Clear, user-friendly interface

### Bonus Features
- 📦 **Batch Conversion** - Convert multiple temperatures at once
- 🔍 **Temperature Comparison** - Compare two temperatures in different units
- 📊 **Quick Reference Table** - Common temperatures for lookup
- 🌍 **Weather Context** - Clothing/activity suggestions based on temperature
- 📐 **Formula Display** - Shows the mathematical calculation

## 📐 Conversion Formulas

| Conversion | Formula |
|------------|---------|
| Celsius → Fahrenheit | °F = (°C × 9/5) + 32 |
| Fahrenheit → Celsius | °C = (°F - 32) × 5/9 |
