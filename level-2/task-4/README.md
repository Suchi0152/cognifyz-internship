# 🌡️ Temperature Converter

## 📋 Project Information
- **Task:** Level 2, Task 4

## 🎯 Objective
Enable users to input temperatures and choose the conversion direction between Fahrenheit and Celsius.

## ✨ Features

### Core Features (Following 4 Steps)
| Step | Feature | Implementation |
|------|---------|----------------|
| Step 1 | Design program to accept temperature input | Scanner with validation |
| Step 2 | Implement conversion logic | `celsiusToFahrenheit()`, `fahrenheitToCelsius()` |
| Step 3 | Allow users to choose conversion direction | Interactive menu |
| Step 4 | Test with different input values | Built-in test function |

### Additional Features
- 📦 **Batch Conversion** - Convert multiple temperatures at once
- 🔍 **Temperature Comparison** - Compare two temperatures in different units
- 📊 **Quick Reference Table** - Common temperatures for lookup
- 🌍 **Weather Context** - Clothing/activity suggestions based on temperature
- 📐 **Formula Display** - Shows the mathematical calculation
- 🧪 **Built-in Tests** - Automated test cases

## 📐 Conversion Formulas

| Conversion | Formula |
|------------|---------|
| Celsius → Fahrenheit | °F = (°C × 9/5) + 32 |
| Fahrenheit → Celsius | °C = (°F - 32) × 5/9 |
